# 职业规划追踪系统

> ⚠️ 本系统为**可选功能**，仅在用户明确要求保留记录或后续跟进时才使用。

## 用户状态文件

保存路径：用户指定目录（由用户明确提供，默认为当前工作目录）

## 追踪原则

- **用户自主控制**：追踪记录的存在和存储位置完全由用户决定
- **数据最小化**：仅保存用户主动提供的关键追踪信息
- **可删除**：用户可随时要求删除已保存的追踪记录
- **不自动创建**：不默认在用户无明确授权情况下写入任何文件

## 追踪节点

- 7天：首次跟进，问候+执行困难
- 30天：月检视，对照目标评估进展
- 90天：季度复盘，决定是否调整方向
- 年度：全面回顾，重新生成规划

## 复盘问卷

1. 上次推荐的行动，你做了哪些？
2. 有遇到什么困难吗？
3. 现在对方向还有信心吗？
4. 有没有发现新机会或新想法？
