# 第三部分：四大功能库详解

你好！欢迎来到第三部分。

现在，让我们深入了解萌芽的四大功能库——这是萌芽区别于普通AI工具的核心能力所在。

掌握这四大功能库，你就能真正发挥萌芽的强大威力，让AI从"聊天工具"变成"生产力助手"。

---

## 3.1 专家库（Agents）

### 什么是专家库？

专家库是萌芽最核心的功能之一。

在这里，你可以：
- 创建和管理各种AI智能体（专家）
- 组建专家团队（主智能体+子智能体）
- 为每个专家配置专属的能力（模型、工具、知识库、技能库等）
- 保存和复用你创建的专家

### 专家库界面介绍

点击左上角的"专家库"，你会看到这样的界面：

![专家库](/docs-images/agent-library.png)

*说明：左侧是"我的创建"和"我的订阅"列表，右侧是选中专家的详细信息*

**左侧边栏**：
- **我的创建**：你自己创建的所有智能体
- **我的订阅**：从智能体市场订阅的他人创建的智能体
- **系统预设**：萌芽提供的默认智能体

**右侧详情区**：显示选中智能体的完整配置信息

### 智能体的配置参数

每个智能体都可以配置以下参数：

![专家配置](/docs-images/agent-config.png)

*说明：展示short_term_fund_filter的详细配置界面*

| 参数 | 说明 | 示例 |
|---|---|---|
| **类型** | 主智能体/子智能体 | 子智能体 |
| **模型** | 使用的AI模型 | kimi-k2.5-thinking |
| **中文名称** | 显示名称 | 短期热点基金筛选助手 |
| **描述** | 这个智能体的用途说明 | 专为两周短期投资设计的基金筛选助手... |
| **系统提示词** | 定义智能体的角色、能力、工作流程 | 你是专业的短期热点基金投资分析师... |
| **工具** | 绑定的工具库 | 可配置该智能体能使用哪些工具 |
| **知识库** | 绑定的知识库 | 可配置该智能体能参考哪些资料 |
| **子智能体** | 下属的子智能体列表 | 主智能体可配置调度的子智能体 |
| **技能库** | 绑定的技能 | 可配置该智能体掌握的技能 |

### 主智能体 vs 子智能体

**子智能体**：
- 专注于某个具体任务的"专员"
- 例如："市场热点侦查员"、"基金数据猎手"、"文案撰写专员"
- 通常只负责整个任务中的一个环节

**主智能体**：
- 负责统筹调度的"组长"或"项目经理"
- 接收用户任务，分配给各个子智能体
- 整合子智能体的输出，给出最终结果
- 可以配置多个子智能体作为"下属"

### 如何创建智能体？

有两种方式：

**方式一：通过专家模式自动生成（推荐）**
1. 新建对话页面时选择"创建专家团队"
2. 描述你的任务需求
3. 萌芽会自动分析并生成专家团队
4. 确认后，所有智能体自动保存到专家库

**方式二：手动创建**
1. 点击"+"按钮新建智能体
2. 填写名称、描述、系统提示词
3. 选择模型类型
4. 配置工具、知识库等参数
5. 保存即可

### 如何使用智能体？

创建好的智能体可以在对话中直接使用：

1. 在底部输入区域，点击"专家库"选择器
2. 选择你想使用的智能体
3. 输入你的问题，该智能体会用预设的配置来回答

**💡 使用场景**：
- 日常通用问题 → 使用默认通用模式
- 专业领域问题 → 选择对应的专家（如"法律咨询专家"）
- 复杂项目任务 → 选择专家团队（如"行业研究报告团队"）

## 3.2 知识库（Knowledges）

### 什么是知识库？

知识库是萌芽的**"资料管理中心"**，适合管理**海量资料**（十几份以上）。

**知识库 vs 直接上传文件**：
- **知识库**：资料多且杂，每次只需搜索其中最相关的片段时使用
- **直接上传**：资料少（1-3份）且高度相关，需要AI完整理解全部内容时使用

知识库的核心优势：
- **一次上传，永久复用**：把常用的资料上传到知识库，随时调用
- **自动命名和描述**：萌芽会自动生成知识库的名称和描述
- **双模式检索**：根据需求选择不同的检索方式

### 知识库界面介绍

点击左上角的"知识库"，你会看到：

![知识库](/docs-images/knowledge-base.png)

*说明：左侧是知识库列表，中间是选中知识库的文件列表，右侧是文件预览区*

**左侧**：所有知识库列表，显示名称和简介
**中间**：选中知识库包含的所有文件
**右侧**：点击文件可在线预览内容（支持PDF、Word、Excel、PPT等多种格式）

### 如何创建知识库？

1. 点击"+"按钮
2. 上传文件（支持批量上传）
3. 萌芽会自动：
   - 分析文件内容
   - 生成知识库名称
   - 生成知识库描述
4. 你也可以手动修改名称和描述
5. 保存即可

### 双模式检索

| 模式 | 原理 | 适用场景 | 优势 |
|---|---|---|---|
| **RAG快速检索** | 基于向量匹配，快速找到相关内容 | 简单信息查询、快速问答 | 响应速度快 |
| **AI知识库检索智能体** | 通过段落摘要索引+生成式AI检索，深度理解内容 | 复杂需求、需要综合分析的问题 | 相当于雇了一个"专属资料管理员"，能完成更复杂的检索任务 |

### 如何使用知识库？

如果你想让某个智能体固定使用特定知识库：

1. 在专家库中编辑智能体
2. 在"知识库"参数中绑定特定知识库
3. 该智能体每次工作都会自动检索这些资料

**💡 使用场景**：
- 客服智能体 → 绑定"产品知识库"
- 法律顾问智能体 → 绑定"法律法规库"
- 行业分析师智能体 → 绑定"行业报告库"

## 3.3 工具库（MCPs）

### 什么是工具库？

工具库是萌芽的**"能力扩展中心"**。

如果把AI比作大脑，工具就是它的"手脚"——让AI能够：
- 搜索互联网信息
- 查询地图和路线
- 获取股票数据
- ...等等

### 工具库界面介绍

点击左上角的"工具库"，你会看到：

![工具库](/docs-images/tool-library.png)

*说明：顶部有SSE工具、HTTP工具等标签，下方是分类折叠的工具列表*

**顶部标签**：
- **SSE工具**：Server-Sent Events协议的工具
- **HTTP工具**：基于HTTP协议的工具
- **个人工具**：你自定义添加的工具
- **系统工具**：萌芽系统内置的工具

**💡 提示**：点击分类可以展开查看该分类下的所有具体工具。

### 工具来源

萌芽的工具来自：
- **ModelScope**：阿里云的模型开放平台
- **腾讯云**：腾讯的MCP市场
- 其他免费开放的MCP平台，持续更新中...

### 智能工具匹配

**你不需要手动选择工具！**

萌芽会：
1. 自动理解你的需求
2. 智能匹配最优工具组合

## 3.4 技能库（Skills）

### 什么是技能库？

技能库是萌芽即将推出的功能，目前正在开发中。

在AI Agent领域，**Skill（技能）**指的是：
- 特定领域的知识包或工作流模板
- 包含完成某类任务的最佳实践
- 可复用、可移植的能力组件

### 技能的价值

想象一下：
- 不需要每次都教AI"怎么写一份合格的合同"
- 不需要每次都解释"怎么做竞品分析"
- 只需要加载"合同审查技能"或"竞品分析技能"

AI就能立即具备该领域的专业能力，按照既定的流程和标准执行任务。

### 萌芽的技能库规划

未来，萌芽的技能库将支持：
- 用户自定义技能包
- 从技能市场订阅他人分享的技能
- 将常用工作流程保存为技能
- 在智能体中绑定特定技能

**💡 当前状态**：技能库功能正在开发中，敬请期待！

## 3.5 四大库如何协同工作？

现在你已经了解了四大功能库各自的作用。让我们通过一个实际案例，看看它们是如何协同工作的。

### 案例：行业研究报告生成

**任务**：生成一份"2024年新能源汽车行业研究报告"

| 功能库 | 作用 | 具体协作内容 |
|---|---|---|
| **专家库** | 组建团队 | **主智能体"报告调度中心"**接收任务，调度下属子智能体：<br>• "行业数据搜集员"——搜索行业数据<br>• "政策分析专员"——分析相关政策<br>• "竞品分析师"——分析主要厂商<br>• "报告撰写员"——整合内容生成报告 |
| **工具库** | 提供能力 | • 行业数据搜集员 → 调用搜索工具、金融数据工具<br>• 政策分析专员 → 调用政策法规查询工具<br>• 竞品分析师 → 调用企业信息查询工具、地图工具<br>• 报告撰写员 → 调用文档生成工具 |
| **知识库** | 提供资料 | 所有智能体共享"汽车行业知识库"（包含历年行业报告、技术白皮书、市场分析资料），使用AI知识库检索智能体深度挖掘资料价值 |
| **技能库** | 未来支持 | 加载"行业研究报告撰写技能"，自动按照标准结构（摘要、市场概况、竞争格局、趋势预测等）生成报告 |

**最终结果**：多个子智能体并行工作，效率翻倍；自动调用各种工具获取最新信息；参考历史资料保证内容深度；生成专业、完整的行业研究报告

### 协同工作示意图

```
用户任务
    ↓
专家库（主智能体）—— 任务分解与调度
    ↓
子智能体1 ←→ 工具库（搜索工具）
子智能体2 ←→ 技能库（地图工具）
子智能体3 ←→ 知识库（行业资料）
    ↓
主智能体 —— 整合输出
    ↓
用户获得完整结果
```
