# 第一部分：认识 Soulprout

## 1.1 Soulprout 是什么？

**Soulprout** 是一个集专业化办公与日常陪伴为一体的全能 Agent 平台，核心能力围绕两个支柱展开：

**Soul 模式** — 你的长期 AI 伴侣。它会记住你的偏好、习惯和行为，持续积累对你的理解，随时通过网页、桌面端或微信/飞书/企业微信/小爱音箱等渠道与你沟通，真正做到"以各种方式陪伴在你身边"。

**专家库** — 你的专属 AI 团队。将你的真实业务逻辑、行业知识沉淀为一个个专家智能体，每个专家拥有独立的模型、工具、知识库和技能配置，主智能体统筹调度多个专家并行协作，把复杂业务流程变成可复用、可迭代的生产力资产。

> Soulprout = 懂你的日常助理 × 你业务的专属专家团队
