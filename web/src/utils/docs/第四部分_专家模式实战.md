# 第四部分：专家模式实战

专家模式是 Soulprout 面向复杂业务场景的核心能力，通过多智能体协作将复杂任务高效完成。

---

## 4.1 什么是专家模式？

面对复杂任务，Soulprout 不会让单个 AI"单打独斗"，而是：

1. **分析任务**：理解需求的复杂度和所需能力
2. **组建团队**：为每个子任务匹配专门的 AI 智能体
3. **并行协作**：多个专家同时工作，效率翻倍
4. **统一整合**：主智能体汇总各方输出，交付完整结果

### 专家模式 vs 通用模式

| 对比项 | 通用模式 | 专家模式 |
|---|---|---|
| **适合任务** | 简单问答、单步骤任务 | 复杂项目、多步骤业务流程 |
| **工作方式** | 单个 AI 处理 | 多个 AI 并行协作 |
| **执行效率** | 串行处理 | 并行处理，效率倍增 |
| **结果质量** | 依赖单一 AI 能力 | 各取所长，综合优化 |
| **透明度** | 看不到过程 | Plan / Tools / SubAgents 全程可见 |
| **可复用性** | 一次性 | 专家团队永久保存，持续复用迭代 |

## 4.2 5 分钟组建专家团队

### 实战演示：组建"基金筛选助手"团队

#### 第一步：描述需求

在对话中输入你的业务需求，例如：

```
我要做一个短期基金的筛选助手，帮我筛选出近期表现好的基金。
```

#### 第二步：交互确认细节

Soulprout 不会立刻执行，而是先和你确认关键细节：
- 筛选周期多长？（1 周 / 1 个月 / 3 个月？）
- 关注哪些行业？（科技 / 医疗 / 全行业？）
- 核心筛选指标？（收益率 / 风险等级 / 规模？）
- 最终输出格式？（表格 / 报告 / 列表？）

完全对齐需求后，再按规划执行——从根源避免"答非所问"。

#### 第三步：AI 自动规划专家团队

Soulprout 展示规划的团队结构：

| 角色 | 职责 | 配置 |
|---|---|---|
| **主智能体** 基金筛选调度中心 | 统筹协调整个筛选流程，整合各专员输出 | 强推理模型 |
| **子智能体1** 市场热点侦查员 | 搜索政策利好、行业风口、热点事件 | web_search 工具 |
| **子智能体2** 基金数据猎手 | 获取基金基础信息与近期表现数据 | 金融数据工具 |
| **子智能体3** 短期动量分析师 | 分析近期技术走势与涨跌动量 | 数据分析工具 |
| **子智能体4** 综合评估员 | 结合热点、数据、分析给出最终推荐 | 强综合分析模型 |

#### 第四步：确认并生成

确认团队结构后，Soulprout 自动：
- 为每个智能体配置最优模型
- 绑定对应工具和知识库
- 生成专业系统提示词
- 设置子智能体调用关系
- 将所有智能体保存到专家库

**无需写代码，无需懂 AI 技术，全程对话完成。**

#### 第五步：立即使用

从"专家库"选择器中找到"基金筛选调度中心"，输入：

```
帮我筛选一下最近适合投资的基金。
```

主智能体自动调度各子智能体并行工作，最终整合输出完整的筛选报告。

## 4.3 专家团队的运作原理

### 架构图解

```
用户
  ↓ 提出需求
主智能体（调度中心）
  ↓ 任务分解
┌─────────┬─────────┬─────────┐
│ 子智能体1 │ 子智能体2 │ 子智能体3 │
│ 市场侦查  │ 数据获取  │ 分析评估  │
│ 并行执行  │ 并行执行  │ 并行执行  │
└─────────┴─────────┴─────────┘
  ↓ 结果汇总
主智能体（整合输出）
  ↓ 最终报告
用户
```

### 关键机制

**主智能体的调度能力**：不是简单的"传话筒"——理解任务全貌、合理分解子任务、根据依赖关系安排执行顺序、对子智能体输出进行质量把控。

**子智能体并发执行**：多个子智能体同时工作，原本串行需要 30 分钟的任务，并行只需 10 分钟。

**全流程透明化**：右侧"工具信息"实时显示每个子智能体的运行状态和工具调用详情。发现问题可随时干预和调整。

**子智能体独立性**：每个子智能体有独立的模型、系统提示词、工具库、知识库和上下文，专注于自己的任务，互不干扰。

## 4.4 自定义你的专家团队

自动生成的团队可以随时调整：

1. 进入"专家库"，找到目标智能体，点击编辑
2. 调整以下参数：

| 参数 | 建议 |
|---|---|
| **模型选型** | 推理任务选强推理模型，创意任务选创意模型，追求速度选轻量模型 |
| **系统提示词** | 细化角色定义，添加专业术语、SOP 流程和输出格式规范 |
| **工具** | 按任务需要精确绑定工具，限定能力边界 |
| **知识库** | 绑定内部资料库，让专家能参考真实业务文档 |
| **技能** | 加载技能包，赋予文档生成、数据处理等专业执行能力 |
| **子智能体** | 调整团队结构，增减子智能体 |

专家团队**永久保存**到专家库，随时调用，持续迭代——从"每次重新招人"升级为"拥有一支常备军"。
