# 第六部分：全面开源与私有化部署

Soulprout 以 **MIT 协议完全开源**，任何人都可以自由使用、修改和分发。在此基础上，我们提供两种部署模式，满足从个人用户到企业团队的不同需求。

---

## 6.1 MIT 开源

Soulprout 采用 MIT 开源协议，这意味着：

- **自由使用**：个人、团队、企业均可免费使用
- **自由修改**：可以根据自己的需求修改代码、添加功能
- **自由分发**：可以基于 Soulprout 构建自己的产品
- **无商业限制**：商业使用同样免费，无需授权

开源地址：见项目 README

我们相信开放带来更好的产品。欢迎提 Issue、提 PR，一起让 Soulprout 变得更好。

## 6.2 两种部署模式

### 模式一：个人服务器模式

类似于 Openclaw 的本地部署方式，**只服务于你自己的设备和账号**。

**适合人群**：个人开发者、技术爱好者、注重数据隐私的个人用户

**核心优势**：
- 数据完全本地化，不经过任何第三方服务器
- 可直接接入本机文件系统和本地服务
- 自由配置模型 API Key，按需选择任意大模型
- 完全掌控运行环境，随时定制扩展

**部署方式**：克隆代码仓库 → 配置环境变量 → 启动服务 → 浏览器访问

---

### 模式二：SaaS 企业内部部署

将 Soulprout 部署到**企业内部服务器**，成为企业专属的 Agent 平台，服务整个团队或部门。

**适合人群**：有内部系统对接需求的企业、希望统一管理 AI 使用的团队、垂直行业用户

**核心优势**：

| 能力 | 说明 |
|---|---|
| **访问内部数据** | 直接连接企业数据库、API、业务系统，Agent 可操作真实业务数据 |
| **品牌定制** | 替换为企业专属 Logo 和 VI，员工打开的是"咱们公司自己的平台" |
| **数据安全合规** | 数据留在内网，满足金融、医疗、政务等行业的安全合规要求 |
| **工具深度定制** | 开发内部 MCP 工具，对接企业 ERP、OA、CRM 等系统 |
| **技能定制** | 将企业 SOP、行业术语、业务规范封装为 Skill，让 Agent 说"行业内的话" |
| **多用户管理** | 统一账号管理、权限控制、使用监控 |

**一句话**：私有化部署的 Soulprout = 平台成熟能力 + 企业内部数据 + 企业专属品牌，成为企业内部生产力的发动机，服务垂直业务域。

## 6.3 企业部署能力详解

### 工具深度定制

公网上的通用 Agent 只能访问互联网公开信息。私有化部署后，Soulprout 可以通过 MCP 工具配置与企业内部系统完全打通：

- **对接内部数据库**：Agent 直接查询核心业务数据，自动生成精准报表
- **对接企业 API**：Agent 调用内部服务，执行实际业务流程
- **对接业务脚本**：Agent 运行企业自定义的业务逻辑

**对比示例**："帮我出一份 Q2 华南区的客户续约分析报告，按公司模板整理，走审批流程提交给部门总监"

| | 公网通用 Agent | 私有化部署 Soulprout |
|---|---|---|
| **数据访问** | 无法访问内部销售数据 | 直接读取内部数据库 |
| **模板使用** | 不知道公司模板格式 | 调取内部文档库模板 |
| **审批流程** | 无法操作内部系统 | 调用 OA 系统 API 自动提交 |
| **最终结果** | "我无法帮你做这件事" | "报告已生成并提交审批，总监审批后会通知你" |

### 技能定制

每个企业都有自己独特的工作流程、行业术语和业务规范。私有化部署支持企业定制专属 Skill：

- 将行业专业术语、业务规范封装为 Skill，让 Agent 不出专业错误
- 将企业标准操作流程（SOP）做成 Skill，自动按规则处理任务
- 新员工使用 Agent 也能做出符合专家水平的工作成果

## 6.4 部署与支持

**自主部署**：技术团队可直接基于开源代码部署，文档齐全，提供技术支持。

**定制服务**：如需品牌定制、工具开发、Skill 开发或一站式部署交付，欢迎联系我们。

| 服务模块 | 内容 |
|---|---|
| **前端定制** | Logo 替换、主题色、界面风格按企业 VI 调整 |
| **工具对接** | 开发 MCP 工具，完成与内部数据库 / API / 脚本的对接 |
| **Skill 开发** | 根据业务需求开发定制化技能包 |
| **部署上线** | 在企业 Linux 服务器上完成部署、测试、上线 |
| **持续支持** | 使用培训、技术支持、迭代升级 |

---

*适合私有化部署的场景*：
- 数据敏感行业（金融、医疗、政务、制造）
- 有大量内部系统需要 Agent 整合的企业
- 希望统一管控全公司 AI 使用的团队
- 通用 Agent 无法满足行业特定需求的垂直场景
