# 第二部分：快速上手

Soulprout 提供三种核心模式，适配从日常助理到专业办公的全场景需求。

---

## 2.1 通用模式

通用模式是最基础的对话方式。你直接在输入框提问，Soulprout 会**自动判断**是否需要调用工具来完成任务。

- 问天气、查新闻、写文案、翻译润色——直接开口即可
- 需要联网搜索、读取文件、执行代码时，自动调用工具，无需手动指定
- 适合一次性问答与轻量任务

**核心体验**：像和一个聪明助理聊天，它会自己判断用什么工具帮你搞定。

---

## 2.2 Soul 模式

Soul 模式是 Soulprout 的完整体，专为**长期陪伴与持续积累**设计。

- **记忆**：主动记住你的偏好、习惯、个人信息和工作风格，越用越懂你
- **上下文压缩**：长对话自动压缩历史，始终保持高效的上下文窗口，不因对话过长而退化
- **多端触达**：网页 / 桌面客户端，以及微信、飞书、企业微信、小爱音箱、Rokid AI 眼镜等渠道，随时随地接入
- **个性化角色**：你可以为它定义名字、性格和说话风格，让它真正成为你的专属助理

**核心体验**：一个长期陪伴你、始终记得你的 AI 伙伴，以你喜欢的方式出现在你的生活和工作中。

---

## 2.3 专家模式

专家模式面向**复杂业务与专业场景**，通过专家库驱动多智能体协作完成高质量任务。

- **构建专家团队**：描述你的业务需求，Soulprout 自动交互确认细节，规划并生成由主智能体 + 多个子智能体组成的专家团队
- **永久沉淀**：专家团队保存到专家库，随时调用、持续迭代
- **并行执行**：多个子智能体同时工作，效率远超单 Agent 串行
- **全程透明**：Plan（规划）、Tools（工具调用）、SubAgents（子智能体状态）全程可见可干预

**核心体验**：把你的业务知识凝结成一支会干活的 AI 团队，复用一次就省一次时间。

---

## 2.4 界面导览

### 整体布局

| 区域 | 说明 |
|---|---|
| **左上角——四大功能库** | 工具库、知识库、专家库、技能库的入口 |
| **中间——对话区** | 与 Soulprout 的聊天记录，包括规划过程和工具调用结果 |
| **底部——输入区** | 发送消息，上传文件，选择模型、工具、知识库、专家 |
| **右侧——工具信息 / 文件库** | 查看执行过程（Plan / Tools / SubAgents）或管理上传文件 |

### 底部输入区功能

| 功能 | 说明 |
|---|---|
| **上传文件（📎）** | 支持 PDF、Word、Excel、PPT、txt、md、py 等多种格式 |
| **模型选择器** | 手动选择本次对话使用的 AI 模型，默认自动匹配 |
| **工具** | 查看并为本次对话绑定工具，通常无需手动选择 |
| **知识库** | 绑定已上传的知识库，让 Soulprout 参考特定资料 |
| **专家库** | 选择已创建的专家或专家团队，直接召唤专业能力 |

### 右侧边栏

| 标签 | 内容 |
|---|---|
| **工具信息** | Plan（任务规划）、Tools（工具调用详情）、SubAgents（子智能体状态） |
| **文件库** | 当前对话中的所有上传文件，支持在线预览和编辑，可直接修改后告知 Soulprout 继续 |
