## 1.1 AI和Agent是什么？

### AI是什么？

AI（人工智能），简单来说，就是让计算机拥有类似人类的"智能"。你可以把它想象成一个**超级聪明的大脑**：

- 它读过互联网上 billions（数十亿）的文章和书籍，知识量远超任何人类
- 它能理解你说的话，还能用流畅的中文回复你
- 它能写文章、编代码、做数学题、翻译语言……

但传统的AI（比如早期的ChatGPT）有一个局限：**它只是一个"大脑"，只能和你对话，不能直接帮你"干活"**。

举个例子：

> 你对AI说："帮我查查明天北京的天气，然后写一封邮件提醒我带伞。"
> 
> 传统AI可能会告诉你："根据天气预报，明天北京有雨。邮件内容如下：……"
> 
> 但你还得自己打开天气网站确认，自己打开邮箱复制粘贴发送。

### Agent是什么？

**Agent（智能体）** 就是在AI大脑的基础上，长出了"手脚"和"工具"的完整助手。

想象一下：

| | 传统AI | AI Agent |
|---|---|---|
| **形象比喻** | 一个知识渊博的顾问，只能给你建议 | 一个能干的助理，不仅能给建议，还能直接帮你办事 |
| **能力** | 只能对话 | 能对话 + 能查资料 + 能操作软件 + 能调用工具 |
| **例子** | "明天北京有雨，记得带伞" | 直接帮你查好天气，写好邮件，甚至帮你发送 |

Agent的核心特点是：**它不仅能思考，还能行动**。

它能——
- 🔍 **自动搜索**互联网上的最新信息
- 📄 **读取**你上传的文件内容
- 🛠️ **调用各种工具**（查地图、买火车票、查法律法规……）
- 📝 **生成和修改**文档
- 🧠 **记住**你们的对话上下文，越用越懂你

简单说：**AI是大脑，Agent是既有大脑、又能干活的完整助手。**
