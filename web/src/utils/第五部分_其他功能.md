# 第五部分：其他功能

除了前面介绍的核心功能，萌芽还有一些贴心的特色功能，专为办公场景优化设计。让我们一起来了解一下！

---

## 5.1 人机协同编辑

### 告别复制粘贴的低效循环

你肯定熟悉这个流程：
1. 在AI网站提问
2. 等生成结果
3. 复制到本地文档修改
4. 再粘贴回AI
5. 再等生成
6. 再复制……

**萌芽直接砍掉所有冗余步骤。**

### 如何使用人机协同编辑？

**第一步：上传文件**

在对话底部点击上传按钮，选择你要编辑的文件。萌芽支持：
- **阅读格式**：PDF、Word、Excel、PPT、txt、md、py等
- **编辑格式**：txt、md、py等文本格式（Office编辑功能即将上线）

**第二步：AI生成内容**

让萌芽基于文件内容生成或修改内容。例如：
```
请根据这份数据，写一份分析报告。
```

**第三步：直接编辑**

AI生成内容后，在右侧"文件库"中打开文件，你可以：
- 直接修改文字
- 调整格式
- 增删内容

**第四步：让AI继续**

改完后，回到对话框说：
```
我改了一下，请按我修改的版本继续完善。
```

萌芽会自动读取你更新后的文件内容，基于你的修改继续工作。

**全程无需切换页面，无需复制粘贴。**

### 使用场景

- **写报告**：AI生成初稿，你修改润色，AI继续补充
- **改代码**：AI写代码，你调试修改，AI继续优化
- **整理资料**：AI整理资料，你调整结构，AI继续完善

## 5.2 交互式规划

### 从根源避免"答非所问"

传统AI（包括很多Agent工具）有个通病：**急于输出**。

即使信息不全，也会强行生成内容。哪怕开启了"深度思考"或"Agent规划"，也是基于AI自己的理解去执行，完全不了解你的真实需求，结果经常跑偏。

### 萌芽的做法：先确认，再执行

遇到复杂任务时，萌芽会先：
1. **梳理信息不全的地方**："我还需要确认……"
2. **明确需求模糊的点**："你说的XX是指A还是B？"
3. **确认输出目标**："你希望最终输出什么格式？"

**完全对齐需求后，再按规划执行。**

### 实际体验

当你说：
```
帮我规划一次旅行。
```

萌芽不会立刻给你行程，而是先问：
- 去哪里？
- 几天几晚？
- 几个人？
- 预算多少？
- 偏好景点还是美食？
- 有什么特殊要求？

等你回答完，萌芽才生成完全符合你需求的行程规划。

### 为什么这很重要？

**节省反复修改的时间**：
- 传统方式：AI生成→不满意→重新描述→再生成→还不满意……
- 萌芽方式：先确认清楚→一次生成→符合预期

**避免方向错误**：
- 如果一开始就理解错了，后面的工作全白费
- 萌芽的"先确认"机制，从根源避免方向错误
