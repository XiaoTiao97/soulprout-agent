如何配置 Rokid 眼镜

首先打开灵珠平台首页：https://rizon.rokid.com/space/home

如果是首次登录，需要进行个人认证，待认证通过后，点击左侧：项目开发 - 三方智能体 - 创建。

填入：

自定义智能体 ID / URL / AK：根据本页生成的内容填写

智能体名称：可自行填写

其他选项根据个人爱好填写，最后点击确认。
